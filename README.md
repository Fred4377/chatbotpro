# 💬 ChatBotPro - Light Sky Blue Messenger UI

![Preview](screenshots/preview.png)

A clean, native-feeling chat messenger application re-themed to a **Light Sky Blue & White** UI that feels fast and lightweight.

## ✨ Humanized & Localized Features
- 🇰🇪 **Kenyan FAQ Suggestions** — Seeded conversation bubbles help users instantly ask local questions like "Do you deliver to Mombasa?" or "Where is your office in Nairobi?".
- ⏳ **Simulated Typewriter Delay** — Chatbot responses feature a typing indicator delay (`...`) to mimic natural human conversation speeds.
- 📱 **System UI Typography** — Uses native system fonts (SF Pro/Segoe UI) to blend seamlessly into modern device interfaces.

## 🛠️ Tech Stack
- **Frontend:** React, HTML5, CSS3, JavaScript (ES6)
